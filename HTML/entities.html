Entities

HTML entities are special characters that are reserved for use in HTML documents. They are used to display reserved characters in HTML documents. HTML entities begin with an ampersand (&) and end with a semicolon (;).

Common Entities

Entity Description
&lt; Less than
&gt; Greater than
&quot; Quotation mark
&apos; Apostrophe
&amp; Ampersand
